Custom Magic Item Cost


Rules

  1. An item can have no more charges than the spell's level that it is imbued with.
  2. Number of Required Spell Scrolls = Number of Charges * Spell Level
  3. Number of Required Spell Slots = Number of Charges * Spell Level
  4. Spell Slot Cost = Cost of one Spell Slot * Number of Required Spell Slots (A table is here)
  5. Spell Scroll Cost = Cost of one Spell Scroll * Number of Required Spell Scrolls (A table is here)
  6. Cost = Spell Slot Cost + Spell Scroll Cost
